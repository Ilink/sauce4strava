The Sauce Live Segment creator generates a segement file from any Strava activity. The file created conforms to the FIT segment file open standard. This file is compatible with supported Garmin devices and may be compatible with other devices that can support side loading of segment files.

More info on garmin side loading can be found here.

NOTE: Wahoo devices do NOT support side loading segment files.
{-infoIcon-} Create a live segment file for use with supported Garmin devices.

The segment name is how this segment appears on your device when you approach the starting location.
