Athlete FTP History

Enter your best estimated functional threshold power in watts. Any time your threshold power changes, you can enter a new value along with the starting date for this new FTP value to be used for TSS, IF and other work related power calculations.

Using this tabulated version of FTP is recommended over Strava's premium feature for FTP because it maintains the proper TSS values for older rides when your FTP changes over time.
